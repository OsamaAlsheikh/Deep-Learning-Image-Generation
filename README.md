# Deep-Learning-Image-Generation
Text-to-Image Generation with Diffusion Models: Built a latent diffusion pipeline using CLIP and contrastively fine-tuned BERT for cross-modal alignment. Implemented UNet denoising with classifier-free guidance and optimized DDIM, LMS, and Karras schedulers in Stable Diffusion. Evaluated 2,200 pairs using FID, LPIPS, CLIP score &amp; cosine similarity.
